# Tienda Electrónica - Proyecto Final

Proyecto final: e-commerce simple (tema oscuro).  
Incluye: listado de productos, carrito dinámico (localStorage), página de carrito y formulario de contacto (simulado).

## Estructura
- index.html
- carrito.html
- contacto.html
- styles.css
- script.js
- data.js
- assets/images/ (colocar imágenes)

## Cómo ejecutar localmente
1. Abrir carpeta en VS Code.
2. Instalar extensión **Live Server** (recomendado).
3. Click derecho en `index.html` → **Open with Live Server**.
4. Probar agregar productos, ver carrito y formulario de contacto.

## Deploy
Subir todo el repo a GitHub y activar **GitHub Pages** (Branch: main, Folder: root) para obtener la URL pública.
